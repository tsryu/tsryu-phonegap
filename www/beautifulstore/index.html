매장찾기 페이지 css 작업

동적으로 작동하는 부분이 많아 css 버그가 많이 생겼지만, 꼼꼼히 다 찾아서 반응형 대응까지 완료했습니다.





기증페이지 알고리즘과 유효성검사의 노하우 습득

기증페이지는 또다른 핵심기능으로, 기증할 수 있도록 도와주는 입력폼 기능이 있습니다. 이 웹폼은 지역과 상자수에 따라 입력하는 폼이 다르기 때문에 굉장히 복잡했는데요. 알고리즘을 그려보고 경우의 수를 완전히 이해한 뒤, 작업해서 무사히 끝낼 수 있었습니다.

유효성검사는 처음에 예외의 상황이 많아서 굉장히 어려웠지만, 나중에는 모두 function으로 등록해서 가져다 쓰는 방식으로 해결했습니다.
